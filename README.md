# DynamicIslandShell
Dynamic island top bar made with Quickshell

# Installation guide

copy this repository to .config folder and make sure u installed quickshell and QtQuick3d plugin

and add this 2 lines to your hyprland config or rewrite it to ur WM

bind = SUPER, M, global, quickshell:powerMenu
bind = SUPER, N, global, quickshell:stressMenu
